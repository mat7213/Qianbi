- **支持主流检测平台定向优化**：知网、维普、万方、格子达等，按目标平台选择对应模式

| 改稿类型 | 能帮你做什么 |

| **插段 / 衔接** | 在两段文字之间智能插入过渡内容 |
| **按导师意见改** | 输入导师评语，针对性修改对应段落 |
| **自定义指令改** | 告诉 AI 你想怎么改，灵活应对各种修改需求 |

### 怎么用？上传 → 选能力 → 改 → 下载

```
上传 Word 文档 → 进入改稿界面 → 选中段落或章节 → 选择改稿类型 → AI 处理 → 下载更新后的文档
```

1. **上传文档**：支持 Word 等常见格式，适合已有初稿、半成品或外部导出的论文
2. **在线浏览 & 选中内容**：像平台内改稿一样，选中需要处理的段落或章节
3. **选择改稿能力**：扩写、缩写、润色、插图、插表、插公式……按需选用
4. **无限次迭代**：不满意继续改，与平台 **免费不限次改稿** 理念一致
5. **下载成品**：改稿完成后导出 Word，继续本地编辑或送检

### 和「上传降 AI」有什么区别？

| | 上传文档改稿 | 上传文件降 AI |
|--|-------------|--------------|
| **主要用途** | 综合改稿：润色、扩写、插图插表等 | 专注全文降低 AIGC 检测率 |
| **操作粒度** | 可选段落 / 章节，能力丰富 | 全文批量处理，平台定向优化 |
| **典型场景** | 导师让改表述、补图表、扩写某节 | 送检前 AI 率整体偏高 |
| **能否配合** | ✅ 可先改稿润色，再降 AI 过检 | ✅ 可先改结构，再全文降 AI |

**建议：改内容走「上传文档改稿」，过检走「上传降 AI」，两步配合效果更好。**

## AI 写论文怎么用？五步走完论文全流程

> 以下流程与官方[《千笔写作 - 使用指南》](https://my.feishu.cn/wiki/NcyFwAE9biSWDLkw5ypcGWf5nlf)一致，上手即会。

```

```

### 第一步：大纲生成 —— 把「想写什么」变成「怎么写」

**① 还没定题目？用「推荐标题」**

告诉 AI 你的研究方向和需求，**一次给出 10 个论文题目**供选择，告别选题焦虑。

**② 选对类型，大纲才更准**

生成前请选择：**学历阶段**（本科 / 硕士 / 博士等）、**文章类型**（毕业论文、期刊论文、开题报告、职称论文等）及**目标字数**。

拿不准类型？选 **「自定义论文」**，生成大纲后再微调。

**③ 已有开题报告或现成大纲？直接上传**

支持上传 **docx、txt** 等格式，或复制内容上传，系统会基于材料快速生成匹配的大纲。

*小提示：正文生成以当前确认的大纲为准，请在生成全文前核对大纲概要是否完整。*

**④ 约 30 秒，千字级详细大纲到手**

通常包含 **6–8 个章节、数十个子章节**，结构完整、层次清晰，可直接进入编辑。

### 第二步：编辑大纲、勾选服务 —— 按你的要求「定制」

**② 章节自由增删改**

觉得章节太多或太少？右侧一键 **增删章节**，拖拽调整结构，直到符合导师或期刊要求。

**③ 勾选增值服务，答辩资料一次备齐**

| 附加服务 | 帮你解决什么 |

| 开题报告 | 研究背景、意义、方法，框架一次成型 |

**Q：千笔 生成的内容可以直接提交吗？**

A：平台生成的是 **范文级初稿**，建议作为写作参考与框架起点，提交前请结合导师要求 **人工核对、补充与润色**，并遵守所在学校的学术规范。

**Q：还没有题目怎么办？**

A：使用「推荐标题」功能，描述研究方向和需求，获取 10 个题目备选。

**Q：大纲可以改吗？**

A：可以。支持在线增删章节、切换二/三级大纲，且大纲 **免费不限次** 重新生成。

**Q：上传的开题报告会在正文里用到吗？**

A：上传文件主要用于 **生成大纲**；正文以你 **最终确认的大纲** 为准，生成前请务必检查大纲概要是否完整。

**Q：已有 Word 论文，能用千笔的改稿功能吗？**

A：可以。使用 **「上传文档改稿」** 功能，上传 Word 后可在文档内选中内容，调用与平台在线改稿 **完全相同** 的能力，包括扩写、缩写、润色、降重、插图、插表、插公式、插代码、按导师意见修改等。

**Q：上传文档改稿和上传降 AI 有什么区别？**

A：**上传文档改稿** 侧重综合编辑（润色、扩写、补图表等），可按段落精细操作；**上传降 AI** 侧重全文批量降低 AIGC 检测率，按知网 / 维普 / 万方 / 格子达等平台定向优化。两者可配合使用：先改稿润色，再全文降 AI 过检。

**Q：只想降 AI，不想写论文，怎么用？**

A：打开千笔 官网，点击顶部 **「降 AIGC / 降 AI」** 入口，上传 Word 文档，选择知网 / 维普 / 万方 / 格子达等平台，一键处理即可。无需走论文生成流程。

**Q：上传降 AI 支持哪些检测平台？**

A：支持 **知网、维普、万方、格子达** 等主流 AI 检测平台，上传前按学校要求选择对应模式即可。

**Q：降 AI 会改乱论文格式吗？**

A：系统主要改写正文表述，**标题、参考文献、专有名词、整体版式尽量保留**，避免改完还要重新排版。

A：千笔 承诺 **改写后 AIGC 率仍不达标可申请退款**（具体标准以官网公示为准）。建议处理后自行送检确认，不满意按规则申请售后。

**Q：降重、降 AI 效果怎么样？**

A：针对知网、维普、万方、格子达等平台持续优化改写策略；支持 **上传全文** 或 **在线片段改写**，具体效果因原文而异，建议改稿后自行检测确认。

**Q：写论文和降 AI 可以一起用吗？**
